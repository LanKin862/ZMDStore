# ZMDStore - 轻量级桌面自动化工具

ZMDStore 是一款基于 **PySide6** 和 **OpenCV** 开发的终末地自动化搬仓库工具。集成了屏幕图像识别和便捷的素材编辑器。

## 快速开始

### 环境依赖

项目基于 Python 3.12+ 开发，主要依赖如下：

```bash
pip install PySide6 opencv-python-headless pyautogui mss numpy
```

### 运行程序

```bash
python ui_main.py
```

## 使用说明

1. **添加素材**：在“添加素材”页面通过编辑器截取目标物品（item）或仓库入口（region）。物品存入 `item` 目录，仓库入口存入 `region` 目录。如果有额外需要，可以自己从wiki上找物品图放入item目录。本代码库中若未额外说明则均为1440p分辨率下所需的素材。
！！！图像分辨率必须和游戏背包内物品框分辨率一致！！！（比如1080p分辨率下，物品框是98x98，1440p分辨率下，物品框是128x128[上述分辨率单指屏幕分辨率]）
2. **配置任务**：
   - 切换到“搬运任务”页面。
   - 选择起点（Begin）和终点（End）图像。
   - 选择要搬运的物品（Item）图像。
   - 设置执行次数。
3. **开始执行**：执行任务最好在一块纯色背景下进行 [推荐在会客室大屏幕后面，避免环境颜色干扰识别。尤其是源矿]，点击“开始执行”后，程序会有 2 秒倒计时，随后自动执行寻找并移动物品的操作。
4. **停止任务**：随时可以点击“停止”或关闭程序来终止自动化流程。

## 打包指南

使用预写的 `build_exe.ps1` 脚本进行一键打包，该脚本会自动应用体积优化策略：

```powershell
.\build_exe.ps1
```
打包好的程序需要和item和region文件夹置于同一文件夹下才能运行
