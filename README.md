# ZMDStore - 轻量级桌面自动化工具

ZMDStore 是一款基于 **Tauri + React + TypeScript** GUI构建 和 **OpenCV** 开发的终末地自动化搬仓库工具。集成了屏幕图像识别和便捷的素材编辑器。

## 快速开始

### 安装依赖

```bash
npm install
```

这会安装所有依赖，包括：

- React 18
- TypeScript
- Vite（构建工具）
- Tauri CLI 和 API

### 运行开发环境

```bash
npm run tauri:dev
```

这个命令会：

1. 启动 Vite 开发服务器（前端热重载）
2. 编译 Rust 后端（第一次会慢一些）
3. 打开你的桌面应用窗口

**开发体验**：修改前端代码时，应用会自动热重载；修改 Rust 后端代码时需要重启应用。

### 打包生产版本

```bash
npm run tauri:build
```

这会生成可发布的安装包，输出在 `src-tauri/target/release/bundle/` 目录。

### 分发包组装

```bash
powershell -ExecutionPolicy Bypass -File .\build_portable.ps1 -SingleFileBridge
```

## 使用说明

1. **初始化素材**：*region*和*public*文件夹下是仓库间切换必须的模版，现已支持自动识别屏幕分辨率自动选择，但如果是异形屏或者带鱼屏，可能会无法使用。

2. **添加素材**：在“添加素材”页面通过编辑器截取目标物品（item）或仓库入口（region）。物品存入 `item` 目录，仓库入口存入 `region` 目录。如果有额外需要，可以自己从wiki上找物品图放入item目录。推荐从森空岛wiki中下载，下载后导入即可直接使用。

3. **配置任务**：
   
   - 切换到“搬运任务”页面。
   - 设置屏幕分辨率（需要让游戏窗口全屏）
   - 选择起点（Begin）和终点（End）图像。
   - 选择要搬运的物品（Item）图像。
   - 设置执行次数。
   
4. **开始执行**：执行任务最好在一块纯色背景下 **[推荐在会客室大屏幕后面，或者传送到会客室然后视角拉到头顶，避免环境颜色干扰识别。]**，并打开背包，点击“开始执行”后，程序会有 2 秒倒计时，随后自动执行寻找并移动物品的操作。

   新增液体运输的功能，现在可以搬各种瓶子了。勾选液体运输模式后，设置液体与容器即可实现不同液体与容器的搬运。

   **另外液体运输模式下也可不设置液体单独设置容器/物品，该模式下匹配的特征会更丰富，如果哪个物品正常模式下匹配不出来或者出错，可以尝试使用该模式进行识别。**
   一般推荐类似于瓶瓶罐罐比较窄的东西可以用这个模式来搬，但正常情况下应该都不会再出问题了。

   
   **另外新增自定义任务列表，如果有多种物品需要搬，可以一次性设置好直接搬完**

5. **停止任务**：随时可以点击“停止”或使用设置的快捷键来终止自动化流程。
