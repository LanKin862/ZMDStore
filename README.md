# ZMDStore - 轻量级桌面自动化工具

ZMDStore 是一款基于 **PySide6** 和 **OpenCV** 开发的终末地自动化搬仓库工具。集成了屏幕图像识别和便捷的素材编辑器。

## 快速开始

### 环境依赖

项目基于 Python 3.12+ 开发，主要依赖如下：

```bash
pip install PySide6 opencv-python-headless pyautogui mss numpy
```

### 运行程序

```bash
python ui_main.py
```

## 使用说明

1. **初始化素材**：*region*和*public*文件夹下是仓库间切换必须的模版，请按照自己屏幕分辨率进行选择。其中均包含了1080p与1440p分辨率所需素材，默认使用的为1440p分辨率所需素材。2160p（4k）显示器可以降分辨率运行也可以按照格式自行在游戏内截图保存也能正常使用（因为我没4k显示器○|￣|_）

2. **添加素材**：在“添加素材”页面通过编辑器截取目标物品（item）或仓库入口（region）。物品存入 `item` 目录，仓库入口存入 `region` 目录。如果有额外需要，可以自己从wiki上找物品图放入item目录。推荐从森空岛wiki中下载，下载后导入即可直接使用。

3. **配置任务**：
   - 切换到“搬运任务”页面。
   - 设置屏幕分辨率（需要让游戏窗口全屏）
   - 选择起点（Begin）和终点（End）图像。
   - 选择要搬运的物品（Item）图像。
   - 设置执行次数。

4. **开始执行**：执行任务最好在一块纯色背景下进行 **[推荐在会客室大屏幕后面，避免环境颜色干扰识别。]**，点击“开始执行”后，程序会有 2 秒倒计时，随后自动执行寻找并移动物品的操作。
   新增液体运输的功能，现在可以搬各种瓶子了。勾选液体运输模式后，设置液体与容器即可实现不同液体与容器的搬运。
   **另外液体运输模式下也可不设置液体单独设置容器，该模式下的匹配为宽容匹配，如果哪个物品正常模式下匹配不出来或者出错，可以使用该模式进行识别。**
   一般推荐类似于瓶瓶罐罐比较窄的东西可以用这个模式来搬。

5. **停止任务**：随时可以点击“停止”或使用设置的快捷键来终止自动化流程。

## 打包指南

使用预写的 `build_exe.ps1` 脚本进行一键打包，该脚本会自动应用体积优化策略：

```powershell
.\build_exe.ps1
```
打包好的程序需要和item和region文件夹置于同一文件夹下才能运行
